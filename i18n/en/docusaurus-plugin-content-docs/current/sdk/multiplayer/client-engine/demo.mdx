This is a collection of game-related demos that you can use when developing your own projects.

## Multiplayer

### Round Robin Battle Demo

This demo uses [Multiplayer](/sdk/multiplayer/features/) and [Client Engine](/sdk/multiplayer/client-engine/) to implement, the implementation language is JavaScript, all server-side and client-side code took about It took about 7 days to implement all server-side and client-side code. The main features are: quick start, character attribute setting, room battle, etc. For details, please click [project address](https://github.com/leancloud/multiplayer-turn-based-game-demo).

### Real-time battle demo

This demo is a streamlined version of the game "Ball Battle", using [Multiplayer](/sdk/multiplayer/features/) to implement, using Cocos Creator (JavaScript) and Unity (C#) to implement, and it took about 8 days. The demo mainly demonstrates the logic related to mobile synchronisation.

For more information click here:

- [Cocos Creator project website](https://github.com/onerain88/BallBattle)

- [Unity project](https://github.com/onerain88/BallBattle-Unity)

## Weakly connected single player game

LeanCloud Anniversary Game is a WeChat game where players can score points by clicking on falling cakes, and the highest scorers can win prizes. The server side mainly uses [cloud engine](/sdk/engine/overview/) and [leaderboard](/sdk/leaderboard/features/) to implement it. For details, please click [project address](https://github.com/leancloud/LeanCloudBirthday).
