This article mainly introduces the rules for sharing the use of Payfire replacement resources. For the games participating in the Fire Plan, you can scan the QR code on the main page of the Fire Plan to share the H5 page of the Fire Plan to third-party platforms and invite other manufacturers' games to join the Fire Plan. For the scanned and shared games, the TapTap platform will also give developers certain rewards according to the sharing effect. The reward rules are as follows

**1„ÄÅRedeeming payroll traffic acceleration rights**

By participating in PayFire's shared games, the platform offers developers the opportunity to redeem PayFire's traffic acceleration rights based on the exposure and invitation data of the shared landing page.

- Taking the natural month as the time cycle, if the number of exposed users of the shared landing page exceeds 100 per month, you can redeem the "traffic acceleration rights" once, and you must redeem it in the same month, and it will be invalid if you haven't redeemed it after the expiry date. The redemption entry is shown in the image below.
- The 'Invitation Code' on the shared landing page can be used to invite 'new third-party games' to join the Fire Plan, and the 'Traffic Acceleration Benefits' will be redeemed according to the number of successfully invited games, and a single invitation code can be used up to 5 times. For example, if you successfully invite 3 new third-party games, you can redeem the Traffic Acceleration Benefits.
  - For example, if you have successfully invited 3 new games from other publishers to join PayFlame, you can redeem the PayFlame Traffic Acceleration Benefits 3 times.
  - The criteria for successful invitations are: new games from other publishers join the PayFire programme and the invitation code of your game is used to complete the redemption of the Traffic Acceleration Benefits for 1 time.
- Traffic Acceleration Benefits redeemed via sharing or invitation code are valid for 6 months and are not subject to the limit of the number of times they can be used in the same month.

**2. Replace traffic resources on "TapTap Resource Replacement Platform".**

For games that participate in the Salary Fire Programme and are shared successfully, the Salary Fire Programme is linked with [TapTap Resource Replacement Platform](https://rep.taptap.cn/), and developers can receive the corresponding traffic rewards at [TapTap Resource Replacement Platform](https://rep.taptap.cn/) according to the rules below.

- For games that have been successfully shared, the "Game Jump Link" on the landing page will automatically generate 1 "Effective Resource" on [TapTap Resource Exchange Platform](https://rep.taptap.cn/), which can be exchanged for related resources according to the access data of the "Game Jump Link".
- Games that complete sharing for the first time can also go to [TapTap Resource Exchange Platform](https://rep.taptap.cn/) to get a "preferred traffic package" worth "RMB1000", which can be received by a manufacturer for a maximum of 5 times.
- If you successfully invite new games from other vendors to join the Fire Plan and become a "new user of TapTap Resource Exchange Platform", you can get the "1,000 RMB Preferred Traffic Pack" again.
- New users of TapTap Resource Exchange Platform refer to vendors who have never received rewards from TapTap Resource Exchange Platform.

**3. Reward Rules for Newly Joined Payfire Games**.

For new games joining the Payfire programme, you can redeem the invitation code at the Traffic Rights and Benefits entrance on the Payfire function page, the entrance is shown below.

- The developer enters the invitation code for redemption, and after successful redemption he/she can get 1 time PayFire traffic acceleration entitlement, which is valid for 6 months.
- If the current publisher is also a new user of [TapTap Resource Exchange Platform](https://rep.taptap.cn/), he/she can also get a newbie traffic gift package worth more than RMB 5,000 on the Resource Exchange Platform.
