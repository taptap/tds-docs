## Icon

### Composition of information

The icon is consisted of the TapTap logo and a background.

### Button Shape

The default shape of the button is circle. It can be changed to a square with rounded corners to match the appearance of other in-game buttons.

### Button size

The default size is 40*40 pt. Please make sure that the aspect ratio of the icon is always 1:1.

### Button margin

The materials in the downloadable resource packs already contain the correct amount of padding, so please do not cut them out or add any extra padding.

### Button size

The default height is 40pt and the button width varies depending on the length of text in different regional languages.

### Button text

Recommended button text is "Login in with TapTap", no line breaks and no spaces between "TapTap".

## Colour rules for login portal

Depending on the background colour of the game interface, two default colours are provided: black, branded (green).

### black

Use this style on a light background that has enough contrast to allow the text to appear in white if you are using a black background for the buttons.

### Rules for customization

As the Login SDK is accessed in a variety of handheld games, game makers may make minor adjustments to the TapTap logo, but must maintain the brand identity of the TapTap logo. There are no restrictions on borders, backgrounds or button text, but changes to the style and colour of the TapTap brand logo are not allowed.

## Rules for the arrangement of multiple login options

The size and style of the login portal background can be adjusted to match the overall style of the game buttons by changing the background of the buttons to any shape, but it must be consistent with the style of the other login forms.
