## 概览
TapTap Payments 是一款为游戏开发者量身打造的支付服务，旨在为开发者在 TapTap 商店销售游戏以及在游戏内销售数字商品和内容提供最佳解决方案。该服务集成了多种付费通道，为全球范围的用户提供便捷的支付体验。

### 开发者的便利

TapTap Payments 提供了简单快速的接入流程，让开发者可以专注于游戏本身而无需花费过多精力在支付系统上。无论是小型独立开发者还是大型游戏工作室，都可以轻松接入 TapTap Payments，并利用 TapTap 开发者中心丰富的功能和工具轻松管理游戏的收入和交易。

### 用户的便捷

对于用户而言，TapTap Payments 提供了全球范围的支付服务，使其可以方便地购买游戏和游戏内的数字商品和内容。不再受限于地理位置或付款方式的限制，用户可以选择他们习惯的付费方式进行交易。

此外，TapTap Payments 提供了高度安全和可靠的支付环境，确保用户的个人信息和交易数据得到保护。用户可以放心地享受游戏内的购买体验，而无需担心安全问题。

 

您可前往 接入文档 了解如何将 TapTap Payments 集成到您的游戏中。在使用 TapTap Payments 之前，请确认您已完成以下事项：

* 已注册成为 TapTap 的开发者

* 成为 TapTap 商家，完成服务签约

* 设置付款资料，便于 TapTap 为您的收入结算付款
 

## 游戏内购买
您可以在游戏内销售各种数字商品和内容（In-App Purchases，以下简称 IAP），为玩家提供更丰富的游戏体验。无论是游戏道具、虚拟货币还是解锁的特殊功能，都可以在 TapTap 开发者中心方便地设置价格和商品信息。

您的游戏通过集成 TapTap Payments  服务，调用 SDK 接口启动收银台，即可实现游戏内购买。用户可以使用多种货币、多种付款方式在游戏内进行快速付款，实现安全高效的交易。

### 商品类型
您可以使用 TapTap Payments 在游戏内销售以一次性付款方式购买，非定期付费的虚拟商品，包括：

* 消耗型商品：是指用户为了获得游戏内容而消耗的商品，通常可以帮助玩家更快地推进游戏进程。商品在被消耗后需要再次购买。例：游戏货币，游戏道具等。当用户消耗商品时，您需要分配关联的收益或效果。

* 非消耗型商品：是指不会过期的一次性付费购买的商品，不会造成消耗，购买后用户可以永久拥有。例：游戏中额外的游戏关卡、应用中无时限的高级会员、游戏皮肤或装备等。

## 支持的用户付款方式
全球国家/地区支持信用卡及多种当地主流钱包支付，包括：Stripe、Paypal、Alipay
