## 1. **图标** <sup><Red>\* 必填</Red></sup>

- 图标尺寸：512x512px
- 图标格式：png/jpg
- 圆角规范：请直接上传直角方图，系统会根据 TapTap 的圆角规范自动裁剪，建议保证主要内容在图示的安全区域内

## 2. **文案**
### 2.1 简介 <sup><Red>\* 必填</Red></sup>

- 内容：描述游戏的类型、玩法以及特色
- 运营建议：建议将主要内容控制在前 50 个字，有利于加深用户对游戏的第一印象

### 2.2 开发者的话

- 内容：描述游戏设计理念、游戏开发心路历程
- 运营建议：建议开发者的话具备个人特色且真情实感，并建议将主要内容控制在前 50 个字

:::tip

- 为避免影响用户端展示效果，简介与开发者的话请勿雷同

:::

### 2.3 更新日志

- 内容：描述本次游戏更新内容
- 运营建议：建议文案简洁明了，使用有序列表展示更新点，比如：1. 2. 3. ...。

![文案](https://capacity-files.lcfile.com/q2AAxaXSLbwOjTdYuOUir13d2y3SabQ2/%E6%96%87%E6%A1%88.webp)

## 3. **视频及截图**

### 3.1. 宣传片视频

- 游戏视频格式：MP4
- 大小：1 GB 以内
- 分辨率：1280 x 720px 及以上，推荐 1920 x 1080px 
- 视频封面尺寸：900 x 600px，格式必须为 JPG 或 PNG 
- 内容：展示游戏的核心玩法为主
- 运营建议：游戏更新后可呈现游戏新版本 / 新角色 / 新玩法素材、节日相关素材等；视频前 3s 代表了游戏的第一印象，请避免开屏的 logo、黑屏等无游戏实机画面的情况

:::tip

- 当有游戏视频时，必须提供视频封面
- 视频还将用于首页智能推荐，视频可选择性提供
- 如果未上传视频，将展示推广图，如无推广图，则展示详情页顶部图

:::

![视频及截图](https://capacity-files.lcfile.com/MFSgXvCI8fa5hmqcVlBbNdJprdDEVWAn/%E8%A7%86%E9%A2%91%E5%8F%8A%E6%88%AA%E5%9B%BE.webp)


### 3.2. 截图<sup><Red>\* 必填</Red></sup>

- 图片方向：支持选择横图、竖图方式展示游戏画面，但请保证每张截图的图片方向一致，如：如果您选择上传横图，请保证所有的图片均为横图
- 横图尺寸：1280x720px 及以上
- 竖图尺寸：720x1280px 及以上
- 格式：png/jpg
- 数量：至少 5 张，请勿上传相同截图
- 大小：建议每张截图 1 MB以内，截图文件大小过大会导致上传速度及用户打开游戏详情页时加载速度过慢，影响游戏转化

:::tip

- 为保证用户能够快速了解游戏玩法，在 TapTap 开放包体下载、测试的游戏截图中必须有实机核心玩法内容展示，实机画面至少占图片的 50%，概念类、广告宣传类图片至多 2 张
- 游戏截图使用游戏中不存在的画面、过度美化游戏中的画面、仅展示卡片、立绘、Loading 界面、主界面等行为，以掩盖游戏本身的真实美术水平、核心玩法，对用户造成误导的，都可能对游戏的正常分发造成影响

:::

![](https://img.tapimg.com/market/images/1e4e0bde2ee71292fc0246bc466dc40c.png)

### 3.3. 真机试玩视频

- 格式：MP4  
- 大小：1 GB 以内  
- 内容：建议提供用户的真实游戏体验录屏

:::tip

- 填写真机试玩视频时，请先提前发布至论坛后，并在提交资料时填写视频链接地址，或者直接填写用户发布的视频链接地址
- 真机试玩视频将用于详情页展示以及随便玩玩专区信息流
- 当未上传视频时，随便玩玩专区将轮播详情页顶部图 

:::

## 4.**详情页顶部图**<sup><Red>\* 必填</Red></sup>

[>> 下载案例文件](https://doc-files.tapimg.com/dev/1920_1080%E5%AE%A3%E4%BC%A0%E5%9B%BE%E6%A1%88%E4%BE%8B.psd)

- 用途：详情页顶部图一般用以突出该游戏的品牌形象，在游戏详情页显示
- 尺寸：1920x1080 px（16:9）
- 格式：png/jpg
- 内容：建议使用游戏主要角色、主要画面场景作为图片的核心视觉，且图片上必须展示清晰可见的游戏名或游戏 Logo

:::tip

- 除了游戏名或游戏Logo外，请勿引用或使用其他文本
- 在某些页面会使用 4:3 比例对图片做裁剪进行展示，请误将游戏 logo、主要角色放置在裁剪线上

:::

![详情页顶部图](https://capacity-files.lcfile.com/Lp6Od3PCzR4sOy8cdFg5EN9ru0CMS0Ou/%E8%AF%A6%E6%83%85%E9%A1%B5%E9%A1%B6%E9%83%A8%E5%9B%BE.webp)


## **5.推广图**

- 用途：用于游戏在资源位展示时使用，如首页推荐、今日游戏等
- 尺寸：1920x1080 px（16:9）
- 格式：png/jpg
- 内容：建议使用最新版本相关素材，并确保及时更新，且图片上必须展示清晰可见的游戏名或游戏Logo

:::tip

- 除了游戏名或游戏Logo外，请勿引用或使用其他文本
- 在某些页面会使用 4:3 比例对图片做裁剪进行展示，请误将游戏 logo、主要角色放置在裁剪线上

:::
