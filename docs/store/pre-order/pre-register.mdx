预约一般在游戏上线前数周或数月开始。完成主要制作、开发、优化、测试，并准备好预约奖励、宣传推广等策略后，您可以在 TapTap 开启预约。

为了吸引更多玩家预约您的游戏，以下是一些有效的运营手段。

## 1. 开启「预约里程碑」功能

### 1.1 什么是「预约里程碑」

预约里程碑适用于帮助游戏吸引玩家进行预约。可在 <Blue>游戏运营</Blue> >> <Blue>内容管理</Blue> >> <Blue>预约里程碑</Blue> 中设置预约里程碑并提交审核，审核通过后将在游戏详情页展示预约里程碑。

![gameadmin](/img/如何吸引-预约里程碑.png)

### 1.2 创建「预约里程碑」有要求吗

1.2.1 预约里程碑内容仅针对游戏正式公测开服。  
1.2.2 预约里程碑支持最大 3 档预约。  
1.2.3 预约里程碑奖励素材尺寸为 200x200 px 透明背景，且大小不超过 100 KB 的 PNG 素材。  
1.2.4 预约里程碑奖励素材不得使用黑白图案。  
1.2.5 预约里程碑里程数仅为 TapTap 平台预约数值。  
1.2.6 预约里程碑里程数需使用阿拉伯数字，不得出现任何符号。  
1.2.7 预约里程碑需对奖励内容进行清晰说明，不得模糊，不可用例如 “ 预约大礼包 ” 等字样表示。  
1.2.8 预约里程碑不可涉及任何实物或金钱性质内容（包括且不限于现金、礼品）。  
1.2.9 预约里程碑发布后不可自行修改，请确认后再提交审核。

### 1.3 如何修改「预约里程碑」

预约里程碑发布后不可自行修改。如因提交错误或不可抗力需修改预约里程碑，请在论坛发布预约里程碑修改公告，公告内容需要包括：新达成奖励数、新奖励内容、预约里程碑需修改原因等。公告发布后，可在 [开发者中心](https://developer.taptap.cn/) >> <Blue>工单</Blue> 中对应的分类下提交修改申请。

### 1.4 如何发放「预约里程碑」奖励

预约里程碑奖励一般在游戏首发后向预约玩家发放，建议在游戏内通过站内信的形式直接向全服玩家发放。

如需通过礼包码的形式发放，请于 游戏运营 >> 激活码&礼包码 >> 新增游戏码 中上传礼包码，并在 工单 联系官方配置发放。

## 2.更新「开发者的话」，与玩家保持交流

![gameadmin](/img/如何吸引-开发者的话.png)

2.1 开发者的话位于游戏详情页的第一屏；展示于视频及截图下方，更新日志（预约里程碑）上方。  
2.2 这是一个可以体现开发者个人风格的信息发布栏，您可以在这里用文字描述设计理念、彩蛋、有趣的文案等内容；首次提交物料后，您还可以多次更新此处内容，与玩家保持互动。  
2.3 开发者的话中文字请勿空行，以免影响在客户端的展示。  

## 3. 举办引导预约活动，并利用「活动中心」扩大曝光

### 3.1 如何在 TapTap 举办一场预约活动

3.1.1 设计预约奖励：您需要制定吸引玩家预约的奖励方案，可以是游戏内物品、虚拟货币、限定角色等，也可以是实物奖品、代金券等。  
3.1.2 确定预约开始及结束时间：您需要确定预约活动的开始时间和结束时间，一般会在游戏上线前的几周或几个月之前开始，以便有足够的时间吸引玩家预约。  
3.1.3 发布并宣传预约活动：论坛发帖后，您需要在论坛、首页、推荐等流量分发位宣传预约活动，以吸引更多玩家关注和预约游戏。  
3.1.4 开启预约：预约活动正式开始后，玩家可以在 TapTap 上进行预约。您可在[开发者中心](https://developer.taptap.cn/)查看预约数据，并根据数据调整推广策略和方案。  
3.1.5 发放预约奖励：预约活动结束后，您需要根据预约数据发放奖励，以回馈预约玩家的支持和信任。  

以上是预约活动的基本流程，您可以根据自己的实际情况和需要进行相应的调整和优化。

### 3.2 如何申请「活动中心」资源位

![gameadmin](/img/如何吸引-活动中心.jpeg)

活动中心是 TapTap 新推出的一个游戏服务栏目，帮助玩家一站式获取游戏招募、礼包、签到、社区活动等福利相关信息，促进用户下载及回流，延长其生命周期。  
当预约活动发布后，您就可以申请活动中心资源曝光支持，申请详情请见[上“活动中心”栏目](/store/events/event-center/)

## 4. 选择转化更高的素材，使用 TapTap 推广中心进行投放

如果想要在游戏预约期间获得更多的关注和用户，使用 TapTap 推广中心进行投放是一个不错的选择。通过 TapTap 推广中心，你可以选择针对特定的目标用户进行投放，从而提高投放效果和转化率。  
同时，为了确保你的广告能够更好地吸引用户的注意力并提高点击率，选择合适的广告素材是至关重要的。通过挑选转化更高的素材，你可以让广告更容易被用户接受和关注，从而增加预约和下载量。记得定期优化和调整广告素材，以便让你的广告始终保持高效和优质。

更详细的说明，可在 [TapTap 广告投放知识库](https://b0yk4osts7.feishu.cn/docs/doccnOIT1vLBnpCWeYtTg1qZjAc?from=from_copylink) 进行查阅。
