确保官方信息更好地触达玩家对于游戏开发者来说至关重要。玩家渴望及时了解游戏更新、活动和重要公告等关键信息，而作为开发者，您有责任确保这些信息准确、及时地传达给他们。

为了让官方信息更好地触达到玩家，以下是一些有效的运营手段。

## 1. 持续运营你的「游戏资讯」

### 1.1 什么是「游戏资讯」

![gameadmin](/img/信息触达1.png)

游戏资讯是为论坛帖准备的详情页曝光位，最多可放置两篇内容，适用于帮助玩家了解最新官方信息。  

游戏资讯的上下架与排序无需审核，开发者可在 <Blue>游戏运营</Blue> >> <Blue>内容管理</Blue> >> <Blue>资讯管理</Blue> 中自助敏捷维护。  

### 1.2 运营「游戏资讯」有哪些须知

1.2.1 游戏资讯仅支持帖子 / 动态 / 视频内容。   
1.2.2 仅本游戏论坛中的内容可被游戏资讯收录。  

## 2.重视「官方帖推送」

### 2.1 什么是「官方帖推送」

![gameadmin](/img/信息触达2.png)

官方帖推送是触达更高的信息传达方式，玩家可在消息中心通过红点提醒获得强感知。  

开发者可在 <Blue>游戏运营</Blue> >> <Blue>内容管理</Blue> >> <Blue>官方帖消息推送</Blue> 中对推送进行管理。  

### 2.2 创建「官方帖推送」有哪些须知

2.2.1 推送时间不可为凌晨 1:00 - 7:00。  
2.2.2 每个自然周最多可发送 1 次推送，次数将于下个自然周重置。  
2.2.3 「定时发送」可选择 5 分钟后的今、明两天内任意时刻定时群发。  
2.2.4 定时推送创建后不可修改，但可在发送前取消。  
2.2.5 被取消的定时推送，不占用本周发送次数。
