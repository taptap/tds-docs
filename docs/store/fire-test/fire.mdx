## 介绍

薪火计划旨在帮助开发者在游戏开发、游戏预热、上线曝光、下载转化等各个阶段降低成本，提高收益。

加入薪火计划，您将获得

- 额外的曝光流量：合作期内，每个自然月均可获得 1 次曝光流量倾斜的机会，自定义使用时间段，预计可增加 30%-50% 的曝光流量

- [免费的 TDS 服务](/sdk)：合作期内，TapTap 开发者中心提供的多项付费服务，包括云存储、云存档、排行榜等全免费

## 如何加入及相关要求

1. 通过开发者中心的「商店配置」-「薪火计划」，自助注册完成加入申请。

2. 游戏必须已接入 TapTap 登录。

3. 合作期内，游戏仅能在 TapTap 独家上线，不可主动提交上架到其他平台，否则 TapTap 将保留追责的权利。

4. 更多细节，可查看 [TapTap 平台薪火计划服务协议](/store/fire-test/fire-service-doc)

## 常见问题及指南

### 薪火计划的额外曝光权益使用攻略

- 加入薪火计划后，可直接获取 3 次额外曝光流量权益，同一个自然月内最多可使用两次加速权益

- 获得权益后如何使用？

  进入薪火计划详情页后，会实时展示当前的权益可用次数，当还有可用次数时，点击「使用流量权益」，进入曝光流量计划预约配置页。

  ![](/img/fire-test/image2022-6-8_17-47-49.png)

  选择预计需要增加曝光的时间段，时间点为选中时间点至之后的六个小时。

  时间段选择完毕后，系统会自动估算当前计划的曝光效果，如曝光效果较差，会向使用者弹出提示告知，此时使用者可选择将计划预约在其他时间段即可继续创建曝光计划。

  ![](/img/fire-test/image2022-6-8_17-52-45.png)

- 什么时候使用曝光权益可以获得更好的效果？

  额外曝光权益是基于游戏原有的曝光权重进行合理加权，提升曝光权重，故在游戏本身的基础流量较高时，可以获得更好效果。建议在游戏首发、重大版本更新、重大运营活动时使用。

### TDS 免费服务的使用攻略

- 自加入薪火计划时刻起至薪火计划的合约周期结束，除涉嫌违规操作外，期间 TDS 的所有服务均免费提供，进行正常接入后即可直接使用。

- 通过工单咨询对应服务，我们的技术支持同事将为您提供全方位的服务。

### 涉嫌违规操作怎么办？

#### 违规操作包括不限于以下行为：

游戏在非 TapTap 的应用平台上架；

同一个加入薪火计划的游戏 ID 被用于多个游戏的服务接入；

游戏内容违反相关法律法规；

#### 涉嫌违规操作会有什么影响？

如游戏涉嫌违规操作，将立即收回流量加速权益的使用权限，在 14 个工作日的处理周期内未按要求完成整改，将停用 TDS 免费服务的使用权，且处理周期内及之后产生的费用将由开发者自行承担，同时对游戏进行下架隐藏处理，并将按照「**TapTap 平台薪火计划服务协议**」中规定的内容，进行追责。

#### 涉嫌违规操作如何整改？

- 如涉嫌上架到其他平台，请尽快联系对应平台的工作人员，下架您的游戏

- 如违规使用 TDS 免费服务，请立即停止超规的使用行为
