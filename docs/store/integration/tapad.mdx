TapAD 是 TapTap 站内推广投放平台，游戏管理者通过 TapAD 创建、优化、管理推广效果，实现推广投放撬动全站价值。大数据算法会把推广内容展现给更感兴趣的用户，帮助不同类型的游戏找到匹配的玩家群体，提升投放效率。  

未来 TapAD 将会在更多原生位置开拓推广位，可选择的投放分端也会逐步完善。  

已开放推广场景：  

 - TapTap 客户端：首页信息流、搜索、标签页、发现页专题  
 - TapTap Web 端：首页信息流、搜索
