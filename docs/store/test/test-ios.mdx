# iOS 测试
作为游戏开发者，您可能希望在正式发布游戏之前测试游戏，以确保游戏质量达到玩家的期望。TapTap 通过 TestFlight 公开链接的方式提供了 iOS 测试能力，帮助您实现这一目标。
在本文档中,我们将解释 iOS 测试是什么以及如何使用它。

# 如何启用此功能​
目前此功能正在白名单测试中，如需使用此功能，请发起工单联系 TapTap 运营开启白名单。

## iOS 测试能够帮助开发者做到什么?

目前在 Android 客户端进行的线上限量测试基本都能够支持，含：接入 TapTap 登录后的内部测试、测试招募、限量资格先到先得、生成短链。

## 如何使用 iOS 测试?
### 一、前期准备 
请先在 TestFlight 中将需要测试的游戏包体添加到测试进程，并生成 TestFlight 公开链接。 
别担心，TestFlight 公开链接只会发送给已获得测试资格或已抢到测试资格的用户。

### 二、如何操作

iOS 测试的所有操作您都可以在 TapTap 开发者中心完成，请参考以下步骤进行:
1. 登录 TapTap 开发者中心，点击商店 >> 版本发布 >> 测试 >> 封闭式测试，创建测试计划。
2. 填写测试计划名称和 TestFlight 公开链接，并保存。
3. 您可根据测试需求选择开启测试招募，如不需要可不操作。
4. 当您需要发放测试资格，或者开放限量资格至用户时，请选择添加测试用户，并按照页面说明进行操作。

更详细的操作演示，请参考帮助文档：
[《内部测试、封闭式测试和开放式测试》](https://developer.taptap.cn/docs/store/test/test-support/#%E4%B8%89%E6%93%8D%E4%BD%9C%E6%BC%94%E7%A4%BA%E5%92%8C%E6%95%88%E6%9E%9C%E6%BC%94%E7%A4%BA-ppt) 。

## iOS 测试暂不支持的能力 

TapTap 的测试需要基于 TestFlight 进行游戏包体的分发，目前 TestFlght 不支持不限量测试，故 TapTap 目前也不支持。

同时，相比较 TestFlight 公开链接更加保密的添加邮箱的方式仍在开发中，目前暂不支持，敬请期待后续更新。
