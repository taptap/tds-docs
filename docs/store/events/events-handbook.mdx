游戏首发上线期期间，开发者可再次申请1次「即将上线」资源位以获得更多曝光，最高平均日曝光可达约3.4w次（实际曝光情况与游戏热度强相关）；此外，首发游戏还将再次自动获得平台首页推荐流量扶持，具体流量倾斜情况依据游戏的预约/下载量变化（预约/下载量越高，给予的曝光扶持越多）。同时，开发者还可以根据指南，利用社区模块发布社区活动，获得更多流量

<br>

## 一、 首发上线期如何获得免费资源位

### 1. 即将上线（需申请）

即将上线是TapTap 新推出的首页栏目，展示当日、前七日及未来较长时间段内 首发、测试以及更新 游戏的信息，提供下载及预约。

更多详细信息与申请指南：[申请「即将上线」栏目，获得免费游戏曝光](https://developer.taptap.cn/docs/store/events/upcoming/)

![](/img/ops-handbook/即将上线.jpg)

<br>

### 2. 发现页：「最新预约」（需申请）
- 申请方式：向对接商务/运营发起申请，审核通过后帮助上线
- 预计曝光：页面每日最高约6万次曝光（实际曝光情况与游戏热度相关）
- 申请时机：游戏页面配置完成前后3个自然日申请
- 申请频次：一次，只支持首次开启预约时申请
- 审核标准：游戏页面素材作为重要参考标准，依据游戏页面素材对玩家的接受度，友好度
- 展示周期：上线后三、四位默认展示一天，之后自然下降（若游戏质量过差或内容违规，则会直接撤出页面）

![](/img/ops-handbook/最新预约.jpg)

<br>

### 3. 首页推荐扶持（自动）
首次开放预约以及首发（预约➡️下载）的游戏，算法会自动帮助进行首页推荐曝光流量倾斜
- 申请方式：无需申请，机器自动配置
- 扶持逻辑：曝光倾斜程度依据游戏的曝光转化率动态调整，相同曝光下，预约/下载的人数越多，算法给予的流量倾斜越多（整个过程算法自动动态调整）
- 扶持时长：整个扶持流程持续3天，3天后进入自然推荐流程，不再做额外倾斜
- 注意：冷启动阶段的流量倾斜仅在首次开放预约和首发各进行一次，版本更新或重复发布无法再次参与

*Tips：详情页的配置丰富度会影响游戏最终的曝光转化率，如何配置优质详情页请见：详情页优化建议*

<br>

### 4. 官方UGC征集活动：新游安利局（需申请）

TapTap为鼓励站内内容创作者为近期上线/大更新的游戏创作内容，增加曝光，定期举办 新游安利局 活动，为内容创作者提供现金奖励。该活动有固定的选游范围，厂商可申请参加。
- 活动举办周期：每15个自然日举办一次（实际举办时间可能会根据实际站内游戏上线情况动态调整）
- 游戏筛选周期：近一个月上线或进行大版本更新的游戏
- 申请时机：请在当期活动开始前至少7个工作日内发起申请，且游戏上线时间须在当期活动的游戏筛选周期内
- 审核标准：编辑根据游戏质量和站内预约量级综合评定
- 申请方式：请与对接运营/商务沟通发起申请
- 案例展示：[新游安利局 Vol.4  0531-0620](https://www.taptap.cn/activity/135)

### 2. 论坛配置及优秀实践分享
