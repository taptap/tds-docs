根据工信部规定，在国内节点绑定的文件域名、云引擎域名、API 域名，都需要备案。

## 新增备案

如果你的主域名没有备案（没有 ICP 备案号），那么主域名本身及其子域名均无法在控制台绑定。
你需要先在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 或其他云服务商办理备案。管局审核通过后，方可在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 控制台绑定。

如果你的主域名没有备案，我们建议你通过 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 新增备案。
主域名在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 备案后，使用子域名在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 绑定文件域名、云引擎域名、API 域名不需要额外备案或接入备案。

<Conditional brand="leancloud">

商用版应用请提交工单，选择 **域名与备案** 下的 **新增备案（企业）** 或 **新增备案（个人）** 分类，按照步骤填写资料，进行新增备案。

没有商用版应用的用户，如果不打算升级应用为商用版，可以通过以下方式新增备案：

1. 如果你同时也是 LeanCloud 底层 IaaS 服务商（华北节点为 UCloud）的用户，那么可以自行在底层 IaaS 服务商新增备案。备案通过后，等价于在 LeanCloud 新增备案。
2. 如果只需要绑定文件域名，可以在其他云服务商处新增备案，然后直接在 LeanCloud 控制台绑定文件域名，因为绑定文件域名不需要接入备案。

</Conditional>

<Conditional brand="tds">

如需新增备案，请提交工单联系我们。
提交工单时，请选择 **游戏服务支持 > 监管政策及相关问题 > 域名备案 > 新增备案** 分类。

</Conditional>

注意，你也可以选择在其他云服务商处新增备案。
但对于大部分用户而言，因为需要绑定 API 域名或云引擎域名，在其他云服务商处新增备案之后，仍然需要在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 接入备案。
这意味着需要重复提交许多类似的资料，经过两次审核周期，由管局先后审核两次，徒然增加时间和精力。
因此，对于主域名没有备案的用户，我们一般推荐在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 新增备案。

## 接入备案

工信部规定，网站接入多个云服务商时，需要在各云服务商处接入备案：

- 接入备案只是在备案信息中新增一个服务商，不会影响之前服务商，可以同时使用。
- 和新增备案不同，接入备案可以在服务上线后进行，不要求关站或停止解析，不影响当前网站访问。
- 一般而言，网站使用的所有云服务商皆需备案或接入备案（在使用的第一家云服务商处新增备案，在其他云服务商接入备案）。

因此，如果你的主域名之前通过 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 新增备案，或者你的主域名已经自行在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 的底层 IaaS 运营商 UCloud <Conditional brand="leancloud">（华北节点）、腾讯云（华东节点）</Conditional>办理过新增备案或接入备案，那么不需要再操心接入备案事项。

如果你的主域名是通过其他云服务商新增备案，那么你在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 控制台绑定 API 域名或云引擎域名后，需要在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 接入备案。
由于 CDN 服务一般不需要接入备案，因此，如果该子域名仅用于 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 文件域名，那么无需在 <Conditional brand="leancloud">LeanCloud</Conditional><Conditional brand="tds">TDS</Conditional> 接入备案。
