:::note
关于如何在控制台上查看日志，以及访问日志等更多内容，请看 [云引擎平台功能 § 查看日志](/sdk/engine/platform/#查看日志)。
:::

云引擎会收集应用打印到标准输出（stdout）和标准错误输出（stderr）的日志：

<EngineRuntimes only={props.only}>
<TabItem value='nodejs'>

```js
console.log("hello"); // stdout
console.error("some error"); // stderr
```

</TabItem>
<TabItem value='python'>

```python
import sys

print('hello') # stdout
print('some error', file=sys.stderr) # stderr
```

<details>
<summary>点击展开 Python 2 打印日志的例子</summary>

```python
import sys

print 'hello' # stdout
print >> sys.stderr, 'some error' # stderr
```

</details>

</TabItem>
<TabItem value='php'>

```php
error_log("some error");
```

</TabItem>
</EngineRuntimes>

:::note
日志单行最大 4096 个字符，多余部分会被丢弃；日志收集速率最大 600 行每分钟，多余的部分会也被丢弃。
:::
