- **定义**

1. TapTap 平台：包括但不限于域名为 com、taptap.io 的网站平台以及由易玩（上海）网络科技有限公司开发或运营的名称为“TapTap”的适用于 iOS、Android 及其他操作系统的移动应用平台。
2. 合作游戏：指加入 TapTap 平台薪火计划，并享受平台流量扶持服务的特定游戏产品。
3. 正式上架：是指您以免费或付费形式在合作区域内正式将合作游戏全面公开给终端用户进行下载使用，同时向用户正式开放游戏内或其他形式的收费服务，收费服务包括但不限于向用户出售道具、虚拟游戏币等形式。

1. **合作期限：自您同意接受本协议之日起至合作游戏在 TapTap 平台正式上架之日起 3 个自然月；如您未在合作期限届满前 1 个自然月内明示 TapTap 取消合作，则本协议将在合作期限届满后按照相同的条件自动续约 1 个自然月，续约次数不限。**
2. 合作区域：中华人民共和国境内。
3. 合作语言：以合作游戏实际提供的语言版本为准。
4. 合作版本：适用于各类终端的非 iOS 操作系统（包括但不限于 Android、鸿蒙和未来上线的操作系统，以及前述所有操作系统的全部版本）的游戏版本。
