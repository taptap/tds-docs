游戏官方号是TapTap为开发者提供的官方认证账号，用于代表开发者官方身份来进行内容发布、粉丝互动和品牌运营。

该功能目前处于邀请测试阶段，将逐步开放，敬请期待。

## 使用游戏官方号的要求

需完善厂商资料，通过平台审核，成为认证开发者。[认证开发者说明](/store/#1-厂商创建通道及流程)

## 游戏官方号的权益

* **认证标识**：获得官方认证标识（蓝V），提升账号的权威性，帮助玩家快速识别官方信息。

* **昵称唯一**：在所有游戏官方号之间不可重名，保护开发者品牌。

* **发布官帖**：无需额外权限，可在关联游戏论坛发布官方帖文，便捷更新官方内容。

* **粉丝共享**：主理游戏后，游戏官方号将获得游戏的粉丝的同步关注。

* **游戏曝光**：游戏官方号的个人主页将展示关联游戏，和置顶游戏。

* **统一管理**：厂商管理员可在开发者中心统一管理所有游戏官方号的账号资料、登录方式，确保所有权归属厂商。

* **邮箱登录**：允许使用邮箱作为游戏官方号登录方式，解决手机号不够用的问题。

## 游戏官方号与游戏的关系

### 主理游戏

游戏官方号主理一个游戏，代表它即是游戏的内容、账号运营主体。它拥有以上所有权益。

它一般与游戏同名，以确保官方性和品牌一致性。

主理关系是一对一的，一个游戏官方号只能主理一个游戏，反之亦然。

例如：主理【火炬之光：无限】的游戏官方号就叫【火炬之光：无限】，负责官方新闻的发布。具体适用场景可参考[游戏官方号运营手册](/store/game_account/operational_guidance/)。

### 关联游戏

游戏官方号关联游戏，代表它参与该游戏的运营，拥有相关的账号定位。它拥有以上除「粉丝共享」以外的所有权益。

它一般使用与游戏有关的名称，以确保关联性。

关联关系是多对多的，一个游戏官方号可以关联多个游戏，反之亦然。

例如：关联【火炬之光：无限】的游戏官方号之一【火炬-暗鸦】，负责官方活动、攻略的发布和论坛互动。具体适用场景可参考[游戏官方号运营手册](/store/game_account/operational_guidance/)。

## 游戏官方号的限制

为保证游戏官方号的官方性，及避免运营中可能出现的账号所有物归属权的争议，游戏官方号在使用中会受到一些限制：

* **账号**：不可修改登录方式（厂商管理员在开发者中心修改除外）、注销账号、授权登录游戏/网站。
* **资产**：不可购买游戏/服务、兑换游戏、获得活动奖励、获取创作者收益。
* **数据**：不可对外展示成就、徽章、头像挂件、游戏时长、战绩、论坛等级、收藏、玩过游戏。
* **功能**：不可评价游戏、云玩、获取测试资格。
* **权限**：不可拥有开发者中心厂商权限。
