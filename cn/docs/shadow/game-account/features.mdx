官方号是 TapTap 上开发者的官方认证账号统称，包含了游戏官方号、制作人员认证账号，用于以开发者官方身份来进行内容发布、粉丝互动和品牌运营。

该功能目前处于邀请测试阶段，将逐步开放，敬请期待。

## 官方号为厂商及游戏带来的核心优势
- **账号轻松涨粉**：游戏官方号绑定游戏，实现新增粉丝共享，账号、游戏双双涨粉。
- **企业统一管理**：实现公私账号隔离，内容和粉丝不再分散在各小编个人账号。
- **带动内容传播**：依托官方号关注关系，内容在动态流、主页更大范围触达目标用户。
- **账号矩阵运营**：自由认证游戏制作人员，配合运营子账号，矩阵式经营官方发声渠道。

## 官方号的分类和使用

![](https://img.tapimg.net/market/images/40cf8587d3b69a9f8429e98bdc02f34e.png)

- 游戏官方号（详见文档 [游戏官方号](https://developer.taptap.cn/docs/store/game-account/official-game/)）
- 制作人员认证号（详见文档 [制作人员认证号](https://developer.taptap.cn/docs/store/game-account/production-staff/)）

他们的主要区别：
- 认证标识：游戏官方号为蓝V；制作人员号为橙V。
- 账号归属：游戏官方号归厂商所有；制作人员号归个人所有。
- 定位：游戏官方号主要以游戏、厂商正式官方身份活动；制作人员号以个人身份活动，更人格化。
- 平台资产：游戏官方号不能用于登录游戏、积累游戏时长和成就等；制作人员号可以像普通账号一样积累平台资产。
